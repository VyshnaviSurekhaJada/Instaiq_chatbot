# InstaIQ 🤖📸

AI-powered Instagram Strategy Assistant built using React, Node.js, and Groq API.

InstaIQ helps users understand Instagram growth strategies, content optimization, engagement techniques, hashtag usage, and algorithm insights through an intelligent chatbot experience.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/VyshnaviSurekhaJada/Instaiq_chatbot.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd instaiq
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Create Environment Variables

Create a `.env` file in the root folder:

```env
GROQ_API_KEY=your_api_key_here
PORT=3001
```

---

## 5️⃣ Run Backend Server

```bash
node server.js
```

Backend runs on:

```text
http://localhost:3001
```

---

## 6️⃣ Run Frontend

Open another terminal:

```bash
npm start
```

Frontend runs on:

```text
http://localhost:3000
```

# ❗ Troubleshooting

| Issue | Solution |
|---|---|
| Failed to fetch | Ensure backend server is running |
| Invalid API key | Verify `.env` configuration |
| Port already in use | Change port in `.env` |
| npm install errors | Delete node_modules and reinstall |
